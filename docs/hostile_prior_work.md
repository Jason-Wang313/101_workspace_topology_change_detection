# Hostile Prior Work

The hostile set contains 100 papers and the manuscript bibliography now draws 230 entries from the local deep-read pool. The strongest threat categories remain:

- Topological SLAM and topology-aware TAMP.
- Dynamic scene-graph planning and scene-change detection.
- Graph neural change classifiers.
- Robust replanning under perception uncertainty.
- Conformal and risk-aware planning.
- Particle-filter belief MPC.
- Neural TAMP and learned affordance-map planners.
- Active view planning for occlusion and topology discovery.
- Robot scene-understanding systems with deployment claims.

The novelty boundary is not "robotics plus more data" or "add uncertainty." The contribution must show that action-conditioned, risk-calibrated topology belief changes what the planner represents, what the planner can choose, and which invalid plans it avoids.

Current v5 local evidence supports that boundary against executable proxy baselines, including topological SLAM/TAMP and dynamic scene-graph transformer proxy. It does not settle the external prior-work threat because no real robot, accepted high-fidelity benchmark, calibrated external topology-change log, or trained checkpoint evidence is present.
